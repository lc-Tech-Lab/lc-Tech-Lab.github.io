Allgemein

Das Wiki ist gespeichert unter technik - Dokumente\Bild-Technik\10_ProWiki

Die Einträge werden als .markdown Dateien gespeichert.

Seite erstellen

Jede Seite wird als eigene .markdown Datei gespeichert. Eine neue Seite kann mit dem MarkText Editor erstellt werden und muss im Ordner 10_ProWiki\markdown gespeichert werden.

Damit die Seite zugänglich wird muss sie auf der Index Seite verlinkt werden. Ein Link wird im Format [Anzeigetext vom Link](Dateiname.html) erstellt. Mit diesem Link kann dann im Wiki die neue Seite geöffnet werden.

Veröffentlichen

Die .markdown Dateien können nicht als Webseite verwendet werden. Nach dem Speichern der Dateien wird die Webseite mit dem Programm convert_md2html im Ordner 10_ProWiki erstellt. Die aktualisierte Version ist dann im Wiki verfügbar.
