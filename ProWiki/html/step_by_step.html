Sonntagmorgen Step by Step

  1. Bildschirme Stage Display und Livestream einschalten

  2. Verknüpfung “Projectors ON” ausführen => startet Beamer

  3. ProPresenter starten, im Planning Center anmelden (Login im Edge Browser)

  4. Im ProPresenter Gottesdienst importieren Linke Spalte bei «Bibliothek» => + => Planning Center Service => GD auswählen => Auswählen warten / dauert etwas

  5. Folien importieren: Stille Infos vor GD, Stille Infos nach GD, Gottesdienstfolien, Predigt (gespeichert unter ../Gottesdienstfolien) Datei => importieren => powerpoint =>Dateien auswählen => ok => importieren in «Presentations» => ok

  6. Folien im GD-Ablauf an den richtigen Ort platzieren: bei Bibliothek auf Presentations klicken, entsprechende Datei mit Maus anwählen und direkt an richtigen Ort im GD-Ablauf ziehen Achtung nicht anklicken, da sich sonst Folien öffnen und GD-Ablauf verschwindet !! Schriften und Darstellung kontrollieren

  7. Bei Stille Infos automatischen Ablauf einrichten: oberhalb Folien rechts auf 1. Zeichen von links klicken => 15 Sec anwählen

  8. Fehlende Songtexte importieren: Linke Spalte bei Bibliothek auf Songtexte => es werden alle Songtitel angezeigt => Filter => Suchfunktion (etwas unterhalb), CTRL + F => Lied mit Maus anwählen und an richtigen Ort ziehen (Achtung ebenfalls nicht anklicken!!) Songtexte auf Fehler kontrollieren (rechte Maustaste, Schnell bearbeiten) Darstellung überprüfen Fenster oben rechts / Textslide to Stream und Saal allenfalls: re Maustaste => Theme => LC Songtexte => Haupttext ohne (ev mit) Übersetzung Falls Anpassung Darstellung nicht klappt => LC Songtexte => Default => ausserhalb Folien klicken -> wieder alle Folien auswählen => LC Songtexte => Haupttext ohne (ev mit) Übersetzung (muss allenfalls mehrmals wiederholt werden, bis es klappt)

  9. Countdown von unten Foreground an entsprechende Stelle im GD-Ablauf ziehen, kann aber auch direkt unten gestartet werden

  10. Nach GD alles ausschalten:

    • sämtliche für diesen GD importierten Folien (Presentations), Gottesdienst (Spalte links oben) und Videos (im Ordner Foreground) löschen

    • Bildschirme Stage Display und Livestream ausschalten

    • Verknüpfung “Projectors OFF” ausführen => schaltet Beamer aus

Fertig :)
