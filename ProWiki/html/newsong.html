Neuer Song

  1. Datei –> Neue Präsentation –> « Liedname », kontrollieren, dass Bibliothek “Songtexte” und Grösse “1920x1080”

  2. Planningcenter in Edge öffnen –> Songs –> Lied suchen –> Lyrics -> Download Datei öffnen –> Text kopieren

  3. Editor öffnen, Text so bearbeiten, dass nur Liedtext, keine überflüssigen Zeilen, Satzzeichen etc. -> kopieren

  4. Auf Folie in ProPresenter einfügen: auf Folie klicken, rechte Maustaste -> Folie bearbeiten -> bei “Text” Doppelklick -> einfügen, Ctrl + Enter –> Text-Trennung auf neue Folie

  5. Style:

    1. Zeilenanfang immer klein

    2. Kein Punkt, kein Komma am Ende der Zeile

    3. Du, Dich, Dein etc. gross – sofern Gott damit gemeint ist

  6. Um Bearbeitung zu verlassen -> Anzeigen

  7. Rechte Maustaste –> gruppieren –> vers/chorus etc. auswählen (es markiert alle folgenden Folien
